pmgversion - Version Info
=========================
:pmg-toplevel:
endif::manvolnum[]

The binary `pmgversion` prints detailed version information for {pmg}
packages.

.Examples

Print {pmg} version:

----
# pmgversion
----

List version details for important packages: 

----
# pmgversion -v
----

Please use the Debian package management for details about other packages

----
# dpkg -l
----
